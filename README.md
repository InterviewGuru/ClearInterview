# ClearInterview

**[Live Website](https://www.clearinterview.in)**

ClearInterview is your ultimate guide to mastering tech interviews. Whether you're preparing for System Design, AI/ML questions, or core Front-End/Back-End concepts, we provide structured resources to help you succeed.

## Features

- **Topic-Based Learning**: Organized sections for System Design, AI/ML, Front-End, and Back-End.
- **Interactive UI**: Clean, modern interface built with Shadcn UI and Tailwind CSS.
- **Deep Dives**: Detailed content on specific topics like API Design, Machine Learning Fundamentals, and React.
- **Responsive Design**: optimized for learning on desktop and tablet.

## Tech Stack

This project is built with:

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Deployment**: GitHub Pages

## Local Development

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/InterviewGuru/ClearInterview
    cd ClearInterview
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment to GitHub Pages.

```bash
npm run deploy
```

This command builds the application and pushes the `dist` directory to the `gh-pages` branch.
